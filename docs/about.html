    About the Computer Exam


    This is the official submission of NeKoRoSYS and Zy for Computer's 3rd Quarter Practical Examination.


  • Lead Project Developer - NeKoRoSYS

    • NeKoRoSYS handled the files structure, the HTML, and CSS code for the whole project.


  • Lead Concept Artist - Zy

    • Zy handled the overall layout, look, and feel for the whole project.


    Features

  • Intuitive and Responsive UI and UX

    • The CSS was developed to provide a user-friendly experience along with a few lines of JavaScript sprinkled on top to make the side navigation work as expected!


  • Clean and Readable Folder Structure

    • This project's source folder was structured in a way to keep things organized and readable. CSS and HTML files are separated, and there is a whole folder dedicated to storing images.


  • Compability and Stability

    • Due to the limited time offered for us to finish this project, we decided that it is best to not rely on third-party libraries to keep the whole project simple and only consist of pure HTML, CSS, and JavaScript. In exchange for stability, we have been bound to some limitations and inconveniences. As much as possible, we also didn't use most latest HTML features and tags to make it compatible to older browsers.



    What We Learned

    We learned that web development requires adequate problem solving skills and basic to intermediate knowledge regarding HTML, CSS, and even JavaScript. NeKoRoSYS and Zy spent a lot of time refactoring old code and designs in order to make the website feel more authentic.
